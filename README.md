# Game Inventory System

Система управления инвентарем для ролевой игры на C#.

## Функционал
- Разные типы предметов: оружие, броня, зелья, квестовые предметы
- Инвентарь с ограничением по весу и вместимости
- Экипировка предметов
- Улучшение предметов
- Unit тестирование

## Паттерны проектирования
1. **Builder** - пошаговое создание предметов
2. **Strategy** - разные алгоритмы улучшения
3. **State** - состояния экипировки
4. **Abstract Factory** - создание наборов предметов

## Технологии
- .NET 8.0
- xUnit для тестов
- SOLID принципы
